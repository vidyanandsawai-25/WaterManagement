## üìã Quick Reference

### Essential Commands
```bash
# First time setup
npm install

# Start development
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Type checking
npm run type-check
```

### Important URLs
- Development: http://localhost:3000
- Alternative port: http://localhost:3001

### Key Files
- Main app: `/App.tsx`
- Components: `/components/`
- Services: `/services/`
- Types: `/types/`
