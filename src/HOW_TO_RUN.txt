QUICK START (3 STEPS):
================================================================================

1. Open Terminal in VS Code
   - Press: Ctrl + ` (backtick key, usually above Tab)
   - Or: Menu > Terminal > New Terminal

2. Install Dependencies
   Type: npm install
   Press: Enter
   Wait: 1-2 minutes for installation to complete

3. Start the Application
   Type: npm run dev
   Press: Enter
   Wait: 10-20 seconds

4. Open Browser
   Go to: http://localhost:3000
   
DONE! Your CRM dashboard should now be running! üéâ

================================================================================
TROUBLESHOOTING:
================================================================================

Problem: "npm: command not found"
Solution: Install Node.js from https://nodejs.org/
          Download LTS version, install, restart VS Code

Problem: "Port 3000 already in use"
Solution: Type: npm run dev -- -p 3001
          Then go to: http://localhost:3001

Problem: Errors during npm install
Solution: 
  Step 1: Delete node_modules folder
  Step 2: Type: npm install
  Step 3: Try again

Problem: TypeScript errors in VS Code
Solution:
  Step 1: Press Ctrl+Shift+P
  Step 2: Type: TypeScript: Restart TS Server
  Step 3: Press Enter

Problem: Module not found errors
Status: Run npm install to install all dependencies ‚úÖ
